# Real Estate Analysis 🏡


## Overview

This project was assigned to me in my MAT-303 course which is Applied Statistics III. I was tasked to analyze a data set that could be used to analyze relationships between different attributes of a house and the house’s selling price. I will create different regression models to predict sale prices for houses. This project showcases skills on creating data visualizations (scatterplots, Q-Q plots) and gathering and analyzing statistics like correlation, summary statistics, and confidence intervals. 

**The main focus for this project is working with second order regression models and the impact quadratic terms have on a model and if they are necessary compared to model without the quadratic terms.**

## Objective
My objective is to create multiple regression models for the housing_v2 data set, using sales price of a house as the dependent variable, and attributes like the size of the living area, number of bathrooms, school ratings, and crime rates as independent variables. Once these models are created, it is my job to analyze the results and determine which model is the most appropriate to predict house prices.
